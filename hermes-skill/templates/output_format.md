## 大乐透格式

```
结果出来了，4策略加权评分，已排除近10期开奖号，推荐如下：
## 🎰 金码银码推荐 & 5注精选
**YYYY-MM-DD | 今日待开：大乐透 XXXXX期**
**🥇 金码：X X X X X + X X**
**🥈 银码：X X X X X + X X**
**📋 5注推荐：**
| 序号 | 前区(5个) | 后区(2个) | 前区和值 | 后区和值 |
|:---:|:---:|:---:|:---:|:---:|
| 1 | **X X X X X** | **X X** | XX | X |
| ... | ... | ... | ... | ... |
---
> ⚠️ 仅供参考，请理性投注！
> 数据来源：大乐透(cwl.gov.cn) 50期官方开奖数据
```

## 格式规则

1. **极简输出**：只展示推荐结果，不展示分析过程
2. **不堆多余内容**：不要统计指标、走势表、遗漏分析等
3. **表格只有4列**：序号、号码、和值、跨度（不要形态列）
4. **免责一行**：⚠️ + 数据来源
5. **福彩3D + 排列三并排**：同一消息输出两个彩种
6. **大乐透单独输出**：前区5+后区2格式
